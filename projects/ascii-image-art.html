ASCII-IMAGE-ART PIP LIBRARY

This is an Open Source simple to use python library which can convert an image to an ASCII text or an ASCII colored images.

This library produces two type of files - 1) a text file which looks an original image but recreated from alphabets and charaters. You optionally provide charaters of your choice as well. 2) an image '.PNG' file which which looks same as the text file but has colors.

There is an additional Machine Learning component which has been added which produces smoother and more compressed results. Using this functionality produces an additional image that is compressed original image.
